# [Chrome Extension] 유튜브 댓글창 증발? 범인은 세로 모니터 & 화면 분할!

**세로 모니터**를 사용하시나요? 혹은 작업 중에 유튜브를 **화면 분할**로 띄워놓고 보시나요?

그렇다면 아마 이 짜증나는 버그를 겪어보셨을 겁니다:
> **"유튜브 라이브 방송을 보다가 일반 영상을 클릭했더니, 댓글창이 감쪽같이 사라져버렸다."**

이건 단순한 오류가 아닙니다. 좁은 브라우저 창을 사용하는 모든 사람에게 발생하는 특정 버그입니다.

## 1. 누구에게 발생하나요? (버그 발생 조건)
이 문제는 다음 특정 조건에서 **100% 발생**합니다.

💡 **영향을 받는 환경:**
1.  **세로 모니터 사용자:** 피벗 모니터를 사용하는 개발자나 디자이너.
2.  **화면 분할 / 멀티태스킹 사용자:** 윈도우 스냅 기능을 이용해 브라우저를 한쪽으로 밀어놓거나 창을 좁게 쓰는 사용자.

**발생 과정:** 유튜브 라이브 시청 (채팅창 활성화) → 추천 영상 등 일반 동영상 클릭.
**결과:** 좁은 화면(뷰포트)에서 브라우저가 "라이브 채팅 레이아웃"에서 "일반 댓글 레이아웃"으로 전환하는 데 실패하여 댓글이 사라집니다.

## 2. 왜 이런 일이 발생하나요?
유튜브는 "반응형 웹"입니다. 브라우저 창 너비가 좁으면(모바일, 세로 모니터, 분할 화면 등) 레이아웃이 변합니다.

문제는 라이브 스트리밍에서 일반 동영상으로 넘어갈 때, 크롬이 *"아직 채팅창 공간을 유지해야 하나?"* 하고 헷갈려 하는 경우가 많다는 점입니다. 이로 인해 댓글창이 로딩되지 않거나 추천 영상 목록에 가려지는 현상이 발생합니다.

## 3. 해결책: YouTube Layout Bug Fix
매번 창 크기를 조절했다 되돌리거나 새로고침(F5)을 하는 것은 매우 번거롭습니다. 그래서 모든 "좁은 화면(Narrow View)" 사용자를 위해 이 문제를 자동으로 해결해주는 확장 프로그램을 개발했습니다.

### 주요 기능:
*   **자동 감지 (Auto Detection):** 세로 모니터, 분할 화면 등 좁은 환경에서의 레이아웃 깨짐을 감지합니다.
*   **자동 복구 (Auto Recovery):** 별도의 조작이 필요 없습니다. 자동으로 댓글창을 즉시 복구해줍니다.

## 5. 설치 방법
이 프로그램은 커스텀 확장 프로그램이므로(스토어 미등록), 다음의 간단한 순서대로 설치해주세요:

1.  위 링크에서 파일을 다운로드하고 압축을 풉니다.
2.  크롬을 열고 주소창에 `chrome://extensions`를 입력한 뒤 엔터를 누릅니다.
3.  우측 상단의 **'개발자 모드 (Developer mode)'** 스위치를 **ON**으로 켭니다.
4.  새로 생긴 **'압축 해제된 확장 프로그램을 로드합니다 (Load unpacked)'** 버튼을 클릭합니다.
5.  압축을 푼 `YouTube Layout Bug Fix` 폴더를 선택합니다.
6.  로드가 완료되면 설치 끝! 이제 댓글창 사라짐 스트레스에서 해방되세요!
